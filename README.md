# jQuery

## Syntax

1. $(): Selector

   1. $(document)
   2. $(`<tag> / <class> / <id>`)
      1. tag - e.g. `$("body")`, `$("h1")`
      2. class - e.g. `$(".btn")`
      3. id - e.g. `$("#level-title")`

2. Listener
   1. .on()
   2. .keypress()
   3. .addClass() / .removeClass()
   4. .text()
   5. chained effect: .fadeIn().fadeOut().fadeIn()
